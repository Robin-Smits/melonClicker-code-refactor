Voor dit project was het de opdracht om een refactor te doen op een open source project, hierbij was het de bedoeling om de code netter te maken zonder de functionaliteit te veranderen. 
Deze refactor heb ik gedaan op een spel wat een klasgenoot had gemaakt in het begin van jaar 1. 
Het over grootte deel van de refactor was duplicatie voorkomen en variabele definitie netter wegzetten.
